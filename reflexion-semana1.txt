La relación entre el uso responsable de la información, la participación ciudadana y la organización del trabajo técnico con Git es muy importante. Cuando consumimos y compartimos información verificada, ayudamos a construir una sociedad mejor informada. La participación ciudadana digital implica opinar, denunciar y proponer con base en hechos reales. Git y GitHub permiten organizar el trabajo técnico de forma colaborativa, llevar un historial de cambios y trabajar en equipo sin perder información. Así, el pensamiento crítico y la tecnología se unen para generar un impacto positivo. Ser ciudadano digital responsable significa también usar herramientas como Git para construir proyectos confiables. Todo esto aporta a la paz digital y social.
